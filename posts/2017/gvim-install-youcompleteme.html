GVIM 安装 YouCompleteMe

星期五, 三月 10, 2017 on Vim, Gvim

YouCompleteMe 是一款功能强大且性能优秀的 VIM 代码补全插件。它支持绝大部分编程语言的代码补全，同时支持代码搜索跳转，使用 VIM 编程的都离不开它。

然而，由于它功能强大，使它的安装没那么轻松。由于在 Linux 环境下安装 YouCompleteMe 相对简单，只需按找文档的安装指南操作即可，本文仅记录 Windows 环境下的安装过程。

系统：Windows 10 （64bit）

环境：GVIM 8.0+（32bit），Python3.3+（32bit），CMake（newest 32bit）， Visual Studio 2015 Community（仅安装 C++ 组件）

需要注意的地方是，所有环境工具（除 VSC）需要安装 32 位版本。

原因是，从 VIM / GVIM （以下统称 VIM）的版本信息（:version）中可以看到

VIM - Vi IMproved 8.0 
...
+conceal            +keymap             +python/dyn         +visual
+cryptv             +lambda             +python3/dyn        +visualextra
+cscope             +langmap            +quickfix           +viminfo

VIM 对 Python 是动态支持（python/dyn），即当 VIM 需要用到 Python 运行环境时才动态加载。不知是否版本的原因，64 位的 VIM 与 64 位的 Python3 无法兼容，导致 YouCompleteMe unavailable: requires Vim compiled with Python (2.6+ or 3.3+) support 的错误。

首先，我们使用 Vundle.vim 或 Git 下载 YouComplete 的版本库（~~由于国情关系~~，建议连上 VPN，否则龟速下载）：

git clone https://github.com/Valloric/YouCompleteMe.git

然后，我们还需下载它的子模块版本库（如果使用 Vundle.vim 安装就不需要再进行此步骤）：

cd YouCompleteMe
git submodule update --init --recursive

下载完成后，我们开始进行编译，在 YouComplete 目录下以启动命令行：

md ycmd_build
cd ycmd_build

cmake -G "Visual Studio 14" -DPYTHON_LIBRARY=E:\Python35\libs\python35.lib -DPYTHON_INCLUDE_DIR=E:\Python35\include -DUSE_PYTHON2=OFF . ..\third_party\ycmd\cpp

-DPYTHON_LIBRARY 与 -DPYTHON_INCLUDE_DIR 分别指向 Python 安装目录的库，按实际进行修改路径。

编译完成后，开始生成项目：

cmake --build . --target ycm_core --config Release

至此，应该已顺利完成 YouCompleteMe 的安装，可启动 VIM 进行测试。
